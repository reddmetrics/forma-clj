forma.utils documentation

average
(average coll)
Returns the average of the supplied collection.

between?
(between? lower upper x)
Returns true of the supplied arg `x` falls between the supplied
`lower` and `upper` bounds (inclusive), false otherwise.

Example usage:
  (between? -2 5 1) => true
find-first
(find-first pred coll)
Returns the first item of coll for which (pred item) returns
logical true. Consumes sequences up to the first match, will consume
the entire sequence and return nil if no match is found.

Example usage:
  (find-first pos? [-1 -2 0 3 4 -1 5]) => 3
force-fill!
(force-fill! buffer stream)
Forces the contents of the supplied `InputStream` into the supplied
byte buffer. (In certain cases, such as when a GZIPInputStream
doesn't have a large enough buffer by default, the stream simply
won't load the requested number of bytes. `force-fill!` blocks until
`buffer` has been filled.
idx
(idx coll)
return a list of indices starting with 1 equal to the length of
input
input-stream
(input-stream arg)(input-stream arg default-bufsize)
Attempts to coerce the given argument to an InputStream, with
automatic flipping to `GZipInputStream` if appropriate for the
supplied input. see `clojure.java.io/input-stream` for guidance on
valid arguments.
moving-average
(moving-average window lst)
returns a moving average of windows on `lst` with length `window`

nth-in
(nth-in coll ks)
Takes a nested collection and a sequence of keys, and returns the
value obtained by taking `nth` in turn on each level of the nested
collection.

Example usage:
  (nth-in [[1 2 3] [4 5 6]] [0 2]) => 3
partition-stream
(partition-stream n stream)
Generates a lazy seq of byte-arrays of size `n` pulled from the
supplied input stream `stream`.
positions
(positions pred coll)
Returns a lazy sequence containing the positions at which pred
 is true for items in coll.

Example usage:
  (positions pos? [-1 -2 0 6 12 -1 2]) => '(3 4 6)
scale
(scale factor coll)
Returns a collection obtained by scaling each element in `coll` by
the supplied number `factor`.

Example usage:
  (scale 4 [0 1 2]) => '(0 4 8)
strings->floats
(strings->floats & strings)
Accepts any number of string representations of floats, and
returns the corresponding sequence of floats.

Example usage:
  (strings->floats "192" "12") => [192.0 12.0]
thrush
(thrush & args)
Very similar to `clojure.core/comp`, this function applies the
function and parameters from left to right in such a way that it
evaluates the function, unlike a macro.  Unlike, `comp` however, this
function accepts an arbitrary number of functions as arguments.

 Example usage:
   (thrush 2 #(* % 5) (partial + 2)) => 12
   (thrush 3 #(* % 5) (partial + 2)) => 17

   similar to: ((comp (partial + 2) #(* % 5)) 2) => 12
trim-seq
(trim-seq bottom top x0 seq)
Trims a sequence with initial value indexed at x0 to fit within
bottom (inclusive) and top (exclusive). For example:

Example usage:
  (trim-seq 0 2 0 [4 5 6]) => [4 5]
unweave
(unweave coll)
Splits a sequence with an even number of entries into two sequences
by pulling alternating entries. For example:

Example usage:
  (unweave [0 1 2 3]) => [(0 2) (1 3)]
weighted-mean
(weighted-mean & val-weight-pairs)
Accepts a number of `<val, weight>` pairs, and returns the mean of
all values with corresponding weights applied. For example:

Example usage:
  (weighted-avg 8 3 1 1) => 6.25
windowed-map
(windowed-map f window-len v)
maps an input function across a sequence of windows onto a vector
`v` of length `window-len` and offset by 1.  Of limited value.

Example usage:
(windowed-map (partial reduce +) 8 (range 10)) => '(28 36 44)
